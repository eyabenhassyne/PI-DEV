# WasteWiseTN – Waste Management Application

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).

The application allows citizens to manage waste recycling and interact with valorization partners.
